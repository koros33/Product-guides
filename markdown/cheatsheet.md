> Markdown is a lightweight markup language for formatting plain text into styled documents.

Quick reference for Markdown syntax.


## Headings
```markdown
# H1
## H2
### H3
#### H4
##### H5
###### H6
````

# H1

## H2

### H3

#### H4

##### H5

###### H6


## Text Formatting

```markdown
*italic* or _italic_  
**bold** or __bold__  
***bold & italic***  
~~strikethrough~~  
`inline code`
```

*italic*
**bold**
***bold & italic***
~~strikethrough~~
`inline code`


## Lists

### Unordered

```markdown
- Item 1
- Item 2
  - Nested Item
    - Sub-nested
```

* Item 1
* Item 2

  * Nested Item

    * Sub-nested

### Ordered

```markdown
1. First
2. Second
3. Third
```

1. First
2. Second
3. Third


## Links & Images

```markdown
[github](https://github.com)

```

[github](https://github.com)

## Blockquotes

```markdown
> This is a blockquote

```

>
> > Nested blockquote
